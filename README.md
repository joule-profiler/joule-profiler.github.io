# Joule Profiler Documentation

Official documentation for Joule Profiler, built with mdBook.

## Overview

This repository contains the source files for the Joule Profiler documentation.
It includes user guides, installation instructions, and technical references.

## Getting Started

### Install mdBook

```bash
cargo install mdbook
```

### Serve documentation locally

```bash
mdbook serve
```

Then open your browser at:

```
http://localhost:3000
```
